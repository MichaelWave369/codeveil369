# CodeVeil369 Safety Boundary

CodeVeil369 is a software simulation, visual art instrument, and claim-safe perception toy model.

It is not a real-world protocol, medical tool, or proof that hidden external code exists in physical reality.

## Hard boundaries

- Keep use inside software simulation and creative visualization.
- Do not point lasers at eyes, people, animals, cameras, vehicles, or reflective surfaces.
- Do not treat generated visuals as evidence of external hidden code.
- Do not use this project for diagnosis, treatment, or mental-health interpretation.
- Do not frame the simulator as proof of simulation theory, metaphysical substrate code, alien communication, or objective hidden writing.

## Intended use

This project is for optical/perceptual toy modeling, symbolic rendering, creative visualization, hypothesis framing, and Ledger-safe exploration of reports involving structured light and perception.

The safest claim is:

> CodeVeil369 simulates how structured light, eye/entoptic filtering, neural-field style pattern formation, and symbolic pattern binding might produce code-like visuals.

That claim is a model, not proof.
