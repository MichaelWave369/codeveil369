# CodeVeil369 Accessibility Notes

CodeVeil369 is visually intense by design. The public demo should stay usable and respectful for a wide range of visitors.

## Current Considerations

- The app uses high-contrast text on a dark background.
- Controls are ordinary HTML buttons, inputs, selects, and file inputs.
- The interface avoids requiring account login or external services.
- Local gallery data stays in the browser.

## Visual Intensity Notice

The field renderer can display animated interference, flicker-like motion, high contrast, and dense glyph patterns. Users who are sensitive to animated visuals should reduce intensity settings, use calmer presets, or avoid fullscreen mode.

## Recommended v1.x Improvements

- Add a reduced-motion preset.
- Add a pause animation control.
- Add clearer keyboard focus styles.
- Add aria labels to the most important action buttons.
- Add a non-visual text summary of the current snapshot state.

## Boundary

Accessibility polish does not change the evidence tier of the project. CodeVeil369 remains a symbolic software simulator and visual instrument.
