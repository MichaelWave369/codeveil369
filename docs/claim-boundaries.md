# Claim Boundaries

## Passed / allowed claims

- CodeVeil369 is a toy-model simulator.
- It renders diffraction-like fields, entoptic-style filters, perceptual amplification, and glyph emergence.
- It can be used to explore how structured visual input might become code-like subjective experience.
- The Material Code Chamber is symbolic/artistic rendering.

## Not passed / forbidden claims

- The app proves reality is made of visible code.
- The app proves a specific altered state reveals source code of the universe.
- The app proves a physical light field contains hidden writing.
- The app is a real-world protocol.
- The app produces medical, psychological, or spiritual diagnosis.

## Recommended public wording

> CodeVeil369 is a claim-safe structured-light perception simulator and symbolic substrate-code renderer.

## Recommended internal Ledger label

> Evidence tier: toy-model / symbolic / artistic. External reality claim: not asserted.
