# CodeVeil369 v1.0 Public Demo Audit

## Status

CodeVeil369 is ready to be treated as a public demo instrument rather than a private prototype.

## Public Boundary

CodeVeil369 is a software visualization and toy-model simulator. It does not claim to prove hidden external code, simulation theory, or a physical substrate language in reality.

## What the Demo Can Claim

- It renders structured-light-inspired visual fields.
- It models entoptic-style filtering as a software layer.
- It provides symbolic glyph and matter-code visualization modes.
- It can save, reload, hash, compare, and export simulator states.
- It preserves receipt language around each snapshot.

## What the Demo Must Not Claim

- It must not claim that the visuals prove external hidden code.
- It must not claim clinical, neurological, or physical validation.
- It must not present itself as a real-world protocol.
- It must not ask users to perform unsafe optical experiments.

## Release Readiness Checklist

- GitHub Pages deploys from Actions.
- Vite base path is configured for `/codeveil369/`.
- Public gallery overlay loads through a relative public asset path.
- Snapshot JSON includes a deterministic hash.
- A/B exports include both current and comparison hashes.
- README and release notes include boundary language.
- Safety documentation is present.

## v1.0 Recommendation

Keep the public landing message simple: CodeVeil369 is a Parallax Lab visual instrument for perception, symbolism, and software receipts. Wonder is allowed; proof requires external evidence.
